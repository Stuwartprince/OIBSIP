# Level 1 - Task 3: Temperature Converter

## Project Name
Temperature Converter

## Description
A responsive web application that converts temperatures between Celsius, Fahrenheit, and Kelvin using HTML, CSS, and JavaScript.

## Features
- Temperature Input
- Unit Selection (Celsius, Fahrenheit, Kelvin)
- Instant Temperature Conversion
- Input Validation
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
